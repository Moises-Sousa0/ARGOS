ARGOS

Argos é um assistente virtual desenvolvido em Python com integração de LLMs locais, reconhecimento de voz e automação por comandos. O projeto começou como um ambiente de estudos para backend, IA local, automação e sistemas, mas a ideia é evoluir para um assistente cada vez mais funcional e inteligente.

Atualmente o Argos consegue ouvir comandos por voz, transcrever áudio em tempo real utilizando Whisper e gerar respostas através de modelos locais executados com Ollama.

Tecnologias utilizadas
Python
Whisper
Ollama
Llama 3.1
SpeechRecognition
CUDA
Funcionalidades atuais
Captura de áudio pelo microfone
Reconhecimento de voz em português
Transcrição utilizando Whisper
Integração com LLM local
Respostas em tempo real no terminal
Encerramento por comando de voz
Tratamento básico de erros
Execução totalmente local
